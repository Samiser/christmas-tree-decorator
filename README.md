# Tinsel Tunes

Play on [itch](https://samiser.itch.io/tinsel-tunes)

A Christmas puzzle game where you decorate trees by placing musical lights to satisfy some puzzling constraints, while creating some festive melodies with bells!

Made for the [Jame Gam Christmas Edition 2025](https://itch.io/jam/jame-gam-christmas-edition).

## How to Play
- Select a light/note colour from the palette at the bottom
- Click on tree slots to place decorations, or drag and drop
- Satisfy the constraints shown on the right to progress
- Press Space to check/submit your solution and hear the melody!

## Controls
- *Left click*: Place selected note
- *Right click*: Remove note
- *Space*: Play/Submit light sequence
